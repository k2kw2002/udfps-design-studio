### 1.1 기본 정보

### 1.2 핵심 문제 정의

OLED 디스플레이 내장 지문인식(UDFPS) 모듈에서 다음과 같은 광학적 문제가 발생하고 있다.

- **AR 코팅 위상 왜곡**: Gorilla DX(4층) 및 DX+(8층) AR 코팅이 입사각(θ)과 파장(λ)에 따라 비균일한 위상 지연 Δφ(θ,λ)를 유발한다. θ=30° 입사 시 DX 코팅 Δφ ≈ -7.5°, DX+ 코팅 Δφ ≈ -12.8°로 측정된다.
- **BM 회절과의 비선형 결합**: Black Matrix(BM) 격자에 의한 회절 패턴과 AR 코팅 위상 왜곡이 비선형적으로 결합하여 점퍼짐함수(PSF)에 비대칭(skewness 0.15~0.35)이 발생한다.
- **지문 위치 오차**: PSF 비대칭으로 인해 지문 인식 위치 오차가 10~30μm 수준으로 나타나며, 이는 인식 정확도를 저하시키는 핵심 원인이다.

### 1.3 기존 한계

기존 광학 설계 도구인 LightTools는 기하광학 기반 ray-trace 방식으로 동작하며, **위상 정보(phase information)를 처리할 수 없다**. 이로 인해 AR 코팅의 위상 왜곡과 BM 회절의 결합 효과를 재현하거나 최적화하는 것이 원천적으로 불가능하였다. 본 과제는 파동광학 기반 물리 정보 신경망(PINN)을 활용하여 이 한계를 극복하는 것을 목표로 한다.

## 2. 기술 스택 및 플랫폼 아키텍처

### 2.1 6-Layer 플랫폼 구조

본 플랫폼은 6개 계층으로 구성되며, 각 계층은 독립적으로 개발 및 확장이 가능하다.

```
┌─────────────────────────────────────────────────────┐
│  L1: Design Studio UI                               │
│      (웹 기반 설계 인터페이스)                         │
├─────────────────────────────────────────────────────┤
│  L2: API / 오케스트레이션 계층                        │
│      (REST API, 작업 스케줄링, 결과 관리)              │
├─────────────────────────────────────────────────────┤
│  L3: Physics Engine ★ (핵심 계층)                    │
│      TMM → ASM → PINN(Helmholtz) → PSF              │
├─────────────────────────────────────────────────────┤
│  L4: 최적화 엔진                                     │
│      FNO Surrogate + BoTorch qNEHVI                  │
├─────────────────────────────────────────────────────┤
│  L5: 데이터 파이프라인                                │
│      LHS 샘플링, 전처리, 결과 저장                     │
├─────────────────────────────────────────────────────┤
│  L6: 인프라                                          │
│      A100 GPU, Python 3.x, PyTorch                   │
└─────────────────────────────────────────────────────┘
```

### 2.2 핵심 모듈 구성

| 모듈 | 역할 | 핵심 기술 |
|------|------|-----------|
| TMM (Transfer Matrix Method) | AR 코팅 위상 해석 | 다층 박막 반사/투과 계수 계산 |
| ASM (Angular Spectrum Method) | Cover Glass 전파 | FFT 기반 파동 전파 시뮬레이션 |
| PINN (Helmholtz) | BM 근방 파동광학 솔버 | SIREN 아키텍처 + 물리 손실 함수 |
| FNO (Fourier Neural Operator) | 실시간 surrogate 모델 | SpectralConv1d 기반 고속 추론 |
| BoTorch qNEHVI | 다목적 최적화 | 3목적 Bayesian Optimization |

### 2.3 UDFPS COE 스택 구조

본 과제에서 다루는 UDFPS 모듈의 물리적 적층 구조는 다음과 같다 (아래에서 위 순서).

```
       ═══════════════════════════════ ← Finger (지문 접촉면)
       ┌─────────────────────────────┐
       │       AR Coating            │  ← Gorilla DX/DX+ (4~8층 박막)
       │  SiO2/TiO2 다층 박막        │     위상 왜곡 Δφ(θ,λ) 발생
       ├─────────────────────────────┤
       │                             │
       │       Cover Glass           │  ← Gorilla Glass (550 μm)
       │       (n = 1.52)            │     ASM 파동 전파 영역
       │                             │
       ├─────────────────────────────┤
       │       Interlayer            │
       ├─────────────────────────────┤
       │       BM2 (Black Matrix)    │  ← 아퍼처 폭 w2 = 40 μm
       │       t_BM = 10 μm         │     회절 격자 역할
       ├─────────────────────────────┤
       │       Interlayer            │  ← BM 간격 d_int = 20 μm
       │       (d_int = 20 μm)      │     PINN 도메인 (파동광학 솔버)
       ├─────────────────────────────┤
       │       BM1 (Black Matrix)    │  ← 아퍼처 폭 w1 = 30 μm
       │       t_BM = 10 μm         │     delta_BM 오프셋 적용
       ├─────────────────────────────┤
       │       Encapsulation         │  ← OLED 봉지층
       ├─────────────────────────────┤
       │       Sensor (Photodiode)   │  ← 지문 영상 취득
       └─────────────────────────────┘
       ═══════════════════════════════

       광 경로: Finger → AR Coating → Cover Glass → Interlayer
               → BM2 → Interlayer → BM1 → Encap → Sensor
```

## 3. 개발 결과

### 3.1 TMM 모듈 (`01_TMM_AR코팅_위상계산.ipynb`)

#### 개발 내용

Transfer Matrix Method를 구현하여 Gorilla DX(4층) 및 DX+(8층) AR 코팅의 반사율, 투과율, 위상 지연을 정밀 계산하는 모듈을 개발하였다.

#### 주요 기능

- Gorilla DX 4층 코팅 구조: SiO2/TiO2/SiO2/TiO2
- Gorilla DX+ 8층 코팅 구조 해석
- 입사각(θ) 및 파장(λ) 의존 위상 지연 Δφ(θ,λ) 계산
- scipy Nelder-Mead 알고리즘을 이용한 층 두께 최적화

#### 달성 결과

| 항목 | 결과 |
|------|------|
| 총 투과율 (T) | > 99% |
| DX 위상 지연 (θ=30°) | Δφ ≈ -7.5° |
| DX+ 위상 지연 (θ=30°) | Δφ ≈ -12.8° |
| 최적화 방법 | Nelder-Mead (scipy.optimize) |

### 3.2 ASM 모듈 (`02_ASM_유리전파.ipynb`)

#### 개발 내용

FFT 기반 Angular Spectrum Method를 구현하여 Cover Glass 내 파동 전파를 시뮬레이션하는 모듈을 개발하였다.

#### 주요 기능

- Cover Glass 두께 550μm에 대한 파동 전파 계산
- FFT/IFFT를 활용한 고속 연산
- TMM 모듈의 위상 출력을 입력으로 수용

#### 시뮬레이션 파라미터

| 파라미터 | 값 | 비고 |
|----------|-----|------|
| 격자 수 (N) | 4,096 | 충분한 해상도 확보 |
| 격자 간격 (dx) | 488 nm | λ/2 이하 (Nyquist 충족) |
| 최대 전파각 (θ_max) | 34.9° | 대상 범위 포함 |
| 위치 오차 | < 0.3 μm | 고정밀 전파 달성 |

Nyquist 샘플링 기준을 충족하여 수치적 안정성을 확보하였으며, 위치 오차 0.3μm 이하의 고정밀 전파 결과를 달성하였다.

### 3.3 Helmholtz PINN (`03_PINN_Helmholtz.ipynb`)

#### 개발 내용

SIREN(Sinusoidal Representation Networks) 아키텍처 기반의 Physics-Informed Neural Network를 구현하여 BM 근방 30μm 도메인에서 Helmholtz 방정식을 풀도록 하였다.

#### 네트워크 구조

| 항목 | 사양 |
|------|------|
| 아키텍처 | SIREN (sin 활성화 함수) |
| 계층 구조 | 4층 x 128 유닛 |
| 총 파라미터 수 | 50,178개 |
| 입력 | (x, z) 좌표 |
| 출력 | 복소 전기장 (Re, Im) |

#### 5항 손실 함수

물리적 제약 조건을 충실히 반영하기 위해 다음 5개 항으로 구성된 복합 손실 함수를 설계하였다.

```
L_total = L_pde + L_bm + L_fresnel + L_data + L_phase★
```

| 손실 항 | 역할 |
|---------|------|
| L_pde | Helmholtz 방정식 잔차 (물리 법칙 준수) |
| L_bm | BM 경계 조건 (회절 격자 조건) |
| L_fresnel | Fresnel 반사/투과 조건 |
| L_data | 관측 데이터 피팅 |
| L_phase★ | AR 코팅 위상 정보 반영 (핵심 혁신) |

#### 훈련 전략

2단계 훈련 전략을 채택하여 수렴성과 정밀도를 동시에 확보하였다.

1. **1단계**: Adam optimizer, 1,000 epochs (전역 탐색)
2. **2단계**: L-BFGS optimizer, 200 steps (국소 정밀화)

μm 단위 정규화를 적용하여 수치 안정성을 확보하였으며, BM delta 최적화를 통해 PSF skewness를 효과적으로 보정하였다.

### 3.4 FNO Surrogate 모델 (`04_FNO_최적화.ipynb`)

#### 개발 내용

PINN의 물리적 정확성을 유지하면서 실시간 추론이 가능한 1D Fourier Neural Operator surrogate 모델을 개발하였다.

#### 네트워크 구조

| 항목 | 사양 |
|------|------|
| 아키텍처 | 1D Fourier Neural Operator |
| 핵심 연산 | SpectralConv1d |
| 정규화 | InstanceNorm |
| 활성화 함수 | GELU |
| 총 파라미터 수 | 531,010개 |

#### 학습 구성

| 항목 | 내용 |
|------|------|
| 샘플링 방법 | Latin Hypercube Sampling (LHS) |
| 학습 데이터 수 | 360 샘플 |
| 옵티마이저 | AdamW |
| 스케줄러 | CosineAnnealing |

#### 다목적 최적화

BoTorch의 qNEHVI(quasi-Noisy Expected Hypervolume Improvement) 알고리즘을 적용하여 다음 3개 목적 함수를 동시에 최적화하였다.

| 목적 함수 | 방향 | 의미 |
|-----------|------|------|
| MTF@ridge | 최대화 (↑) | 변조 전달 함수 (영상 선명도) |
| T_total | 최대화 (↑) | 총 투과율 (광량 확보) |
| \|skewness\| | 최소화 (↓) | PSF 비대칭도 (지문 위치 정확도) |

### 3.5 GS 위상 추정 (`05_GS_위상추정.ipynb`)

#### 개발 내용

Gerchberg-Saxton 알고리즘을 구현하여 강도 분포로부터 위상 정보를 역추정하는 보조 도구를 개발하였다.

#### 평가 결과

Gaussian 빔 조건에서 수렴 한계가 확인되었으며, 이는 AR 코팅의 복잡한 위상 구조를 단순 반복법으로는 정확히 복원하기 어려움을 의미한다. 이러한 한계가 PINN 기반 접근법의 필요성을 뒷받침하는 근거가 되었으며, GS 알고리즘은 초기 위상 추정 등 보조 도구로 활용하기로 하였다.

### 3.6 통합 파이프라인 (`06_통합_파이프라인.ipynb`)

## 5. 핵심 기술 성과

### 5.1 위상 정보 기반 설계 방법론 확립

기존 LightTools ray-trace의 근본적 한계인 위상 정보 처리 불가 문제를 TMM + ASM + PINN 결합으로 극복하였다. AR 코팅의 위상 왜곡 Δφ(θ,λ)를 정량적으로 계산하고, 이를 파동 전파 및 회절 시뮬레이션에 직접 반영하는 end-to-end 설계 방법론을 확립하였다.

### 5.2 PINN + FNO 하이브리드 아키텍처

- **PINN (Helmholtz SIREN)**: 물리 법칙(Helmholtz 방정식)을 손실 함수에 직접 내장하여 물리적 정확성을 보장한다.
- **FNO Surrogate**: PINN의 결과를 증류(distillation)하여 1ms 이하의 실시간 추론을 가능하게 한다.
- 이 하이브리드 구조를 통해 물리적 정확성과 계산 속도를 동시에 확보하였다.

### 5.3 다목적 동시 최적화

MTF(영상 선명도), 광량(T_total), 비대칭도(skewness)의 3개 목적 함수를 BoTorch qNEHVI로 동시에 최적화하는 프레임워크를 구현하였다. 이는 단일 목적 최적화로는 달성할 수 없는 Pareto 최적해 탐색을 가능하게 한다.

### 5.4 BM 오프셋 보정 기법 도출

PSF 비대칭의 원인을 AR 코팅 위상 왜곡과 BM 회절 효과로 분리하고, BM1 오프셋(delta_BM = -20μm)을 통해 비대칭을 보정하는 방법을 제시하였다. 이는 향후 실제 공정에 직접 적용 가능한 설계 지침이다.

## 7. 결론

### 7.1 Phase 1 성과 요약

Phase 1(M01~M06)에서 다음과 같은 핵심 성과를 달성하였다.

1. **24개월 최종 목표 조기 달성**: PSF skewness 지표에서 최종 목표(< 0.10)를 Phase 1에서 달성하였다 (달성값: 0.075).
2. **물리 정보 기반 AI 설계 플랫폼 핵심 모듈 구현 완료**: TMM, ASM, PINN(Helmholtz SIREN), FNO Surrogate, BoTorch 다목적 최적화 등 모든 핵심 모듈의 구현을 완료하고 통합 파이프라인으로 연결하였다.
